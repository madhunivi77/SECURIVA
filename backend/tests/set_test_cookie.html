🔐 Set Test API Key Cookie

Purpose: This page sets a test API key cookie in your browser, allowing you to test Salesforce authentication without going through Google OAuth.

Steps:

  1. Run: uv run python tests/create_test_user_for_salesforce.py
  2. Click "Set Test Cookie" button below
  3. Click "Test Salesforce Login" to authenticate with Salesforce
  4. Run your test scripts

After Setting Cookie:

  • Authenticate Salesforce
  • Then run: uv run python tests/test_salesforce_mcp.py
